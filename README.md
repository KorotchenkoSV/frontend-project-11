# RSS Reader

Сервис для агрегации RSS-потоков. Позволяет добавлять неограниченное количество RSS-лент, автоматически обновляет их и показывает новые записи в общем потоке.

- Vite — сборщик фронтенда
- Bootstrap 5 — CSS-фреймворк
- Чистый JavaScript (с использованием промисов)
- DOM API
